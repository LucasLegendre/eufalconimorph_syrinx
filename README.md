# eufalconimorph_syrinx
Code, data, and phylogenetic trees used for a project on the syrinx of eufalconimorphs, published in Legendre and Clarke (2026). 

This project includes:
- 'Eufalconimorph_project.Rmd': R script for the project, in Markdown format
- 'Euf_data.csv': whole dataset for syringeal traits (see Main Text and Supplementary Table 1 for more information), in CSV format
- 'euf_tree.trees.nex': phylogenetic tree for 'Euf_data.csv', pruned from McTavish et al. (2025), in NEXUS format
- 'ASR_IM.xlsx': whole dataset for presence/absence and number of intrinsic syringeal muscles (see Main Text and Supplementary Table 2 for more information), in XLSX format

The file 'summary_dated_clements.nex', which is the full tree of McTavish et al. (2025) necessary to replicate the ancestral reconstructions for intrinsic muscle traits using 'ASR_IM.xlsx', is available at <https://github.com/McTavishLab/AvesData/blob/main/Tree_versions/Aves_1.3/Clements2023/summary_dated_clements.nex>.
